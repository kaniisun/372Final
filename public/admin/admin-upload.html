File Upload Instructions

Only JSON (.json), CSV (.csv) and TXT (.txt) formats will be accepted.

Fields must include:

  • Product ID
  • Product Name
  • Category
  • Price
  • Description
  • Image Path

Example JSON Format:


        {
            "product_id": 1,
            "product_name": "Bouquet 1",
            "category": "Mother's Day",
            "price": 17.99,
            "description": "Beautiful arrangement of lilies",
            "image": "/images/flower1.webp"
        }

Example CSV Format:


product_id,product_name,category,price,description,image
1,Bouquet 1,Mother's Day,17.99,Beautiful arrangement of lilies,/images/flower1.webp
